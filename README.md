# sangoku-hiphop-guide

三国志HIPHOP制作教科書 — AI音楽クリエイター向けの公開ノウハア集。

MiniMax music-2.6 等の生成AIで三国志武将のHIPHOP楽曲を、武将選択から曲+歌詞+画像+動画まで作る完全ノウハウ。発音・韻・楽曲構造・アートワーク・動画の実践知見。

## ローカル確認
`index.html` をブラウザで開く（外部依存なし・単一ファイル）。

## GitHub Pages
https://fukukei23.github.io/sangoku-hiphop-guide/

## 構成（9章・アコーディオン）
1. はじめに
2. 武将を選ぶ
3. 楽曲を設計する
4. 歌詞を書く ①発音
5. 歌詞を書く ②韻
6. 楽曲を生成する
7. アートワーク
8. 動画を生成する
9. 実例: 曹仁「樊城の守り」

## 出典
`sangoku-song` スキル（claude-config repo・非公開）の制作ノウハウを外向きに再構成。
